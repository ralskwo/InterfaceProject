#자료조사
* 공주 정보
  - 이름, 나이, 국적(지도에 표시할 좌표), 성격, 배경스토리 
* 공주 대사
  - 3개씩
* 공주 관련 컨텐츠
    * 명장면(캡쳐사진?or유튜브 링크?)
    * 노래(유튜브 링크)
    * 관계도?
      - 관련 등장인물
    * 굿즈 사이트?
* 사진
* 공주 목록
  - 김민규
    - 안나
    - 인어공주(에리얼)
    - 잠자는 숲속의 공주(오로라)
  - 성보람
    - 미녀와 야수(벨)
    - 티아나
    - 신데렐라
    - 엘사
  - 임우중
    - 백설공주
    - 모아나
    - 뮬란
    - 알라딘(자스민)
  - 정동규
    - 메리다
    - 포카혼타스
    - 라푼젤
    - 에스메랄다
